**8 ideas universales (combinando 2+ tipos):**

2. **Cross-Chain Arb Router** — Swap agg + Bridge + Market data. Detecta spreads del mismo activo entre cadenas, ejecuta bridge + swap dual-leg si neto > fees+slippage. Alto.
3. **Treasury Co-Pilot** — Liquid staking + Lending + Analytics. Distribuye tesorería entre LSTs, lending y reservas según política riesgo declarada. Reasigna cuando rendimientos comparativos cambian. Medio-Alto.
4. **Launch Sniper con Risk Gate** — Token launch + Oracle + Analytics. Monitorea bonding curves, filtra por liquidez/holders/patrones rug, entra solo si score pasa filtro. TP/SL automáticos. Alto.
5. **Confidential Payroll Engine** — Confidential transfers + Payments + Oracle. Convierte balance estable en pagos recurrentes con transferencias confidenciales y FX vía oracle. Calendariza/firma cada batch. Medio.
6. **NFT Floor Maker** — NFT metadata + Swap agg + Oracle. Lista/cancela NFTs alrededor del floor dinámico, compensa inventario con swaps a estable. Reajusta spreads según volatilidad. Alto.
7. **On-Chain Compliance Watchdog** — RPC streaming + Analytics + Multisig. Vigila wallets de un equipo en tiempo real, detecta interacciones anómalas, propone vetos vía multisig. Pausa fondos si supera score crítico. Medio.
8. **Liquidation Hunter** — Lending + Oracle + Swap agg. Indexa posiciones near-liquidation, calcula PnL bruto post-fees, dispara liquidación + swap del bono recibido. Compite por bloque vía priority fee adaptativo. Alto.

**Gap real:** **Reputation/identity & social-graph capabilities** — DIDs/credentials verificables, grafos sociales on-chain, scoring reputación cross-protocol, gating basado en historial. Hay analytics de wallets como observación pasiva, pero no una capa accionable de "quién es esta wallet y qué puede hacer". También **governance** (propuestas, votación, ejecución DAOs) y **MEV-aware execution** (Jito bundles, sandwich-protection, private mempool routing) ausentes como tipo genérico.

## Agente v3-8 — Web search trending neutral

**7 tendencias reales:**

1. **Pagos agénticos sobre stablecoins (x402) se consolidan como infraestructura.** ~165M tx, ~$50M volumen, ~69k agentes activos. AWS, Coinbase, Stripe, Visa, Mastercard, Linux Foundation respaldan.
2. **Agentes como "primary users" on-chain.** >68% protocolos DeFi nuevos Q1 2026 incluyeron agente autónomo. >250k agentes activos diarios. Más tx iniciadas por software que humanos.
3. **Multi-agent frameworks (Bull/Bear/Risk Supervisor) superan LLMs solos.** 95% hedge funds migraron a multi-agente. Research BlackRock/Columbia confirma.
4. **KYA emerge como problema crítico.** Agentes ejecutan contratos sin marco identidad. Primeros estándares on-chain.
5. **Backlash sobre "AI agent coins" tokenizados.** Virtuals -86% wallets, SIREN -93%. Memecoins agénticos pierden tracción frente a utility real.
6. **Brecha de seguridad LLM routers/wallets agentes.** >$45M pérdidas. 35% empresas no puede apagar agente rogue.
7. **Agentic commerce A2A en producción.** Anthropic experimento 186 deals reales, Google A2A en >150 orgs. Consenso "MCP + A2A + AP2/UCP".

**Clases que SI funcionan vs FALLAN (2026):**

**SI funcionan:**
- Trading/yield/portfolio rebalancing autónomo
- Compliance y monitoring de wallets
- Agentes UX / "intent translators"
- Pagos M2M (x402) sub-cent
- Governance summarization/triage en DAOs

**FALLAN:**
- AI agent memecoins / "agent shares" especulativos
- Agentes "personality/influencer" en X (Lobstar Wilde, Moltbook: pérdidas por ingeniería social y memory leaks)
- DeFi exploit agents autónomos
- Agentes sin circuit breakers ni KYA
- "Agentic payments" sin demanda real validada (infra adelantó al uso)

**Patrón claro:** lo que funciona resuelve un dolor económico medible. Lo que falla son agentes para narrativa/token sin guardrails operativos.
